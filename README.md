# React Pizza

> Полнофункциональное SPA-приложение для заказа пиццы на React и TypeScript.  
> Асинхронные запросы, фильтрация, маршрутизация, управление состоянием и оптимизация производительности.

---

## Возможности

- **Фильтрация и поиск**
- **Пагинация**
- **Асинхронная загрузка данных**
- **Redux Toolkit для управления состоянием**
- **Синхронизация фильтров с URL**
- **Корзина с сохранением в LocalStorage**
- **Динамические маршруты** `/pizza/:id`
- **Оптимизация ререндеров**
- **Code Splitting**

---

## Технологии

- React
- TypeScript
- Redux Toolkit
- React Router
- SCSS / CSS Modules
- Axios
- Lodash.debounce
- React Hooks
- React.lazy

---

## Архитектура

- Feature-подход к организации Redux-слайсов
- Асинхронная логика через `createAsyncThunk`
- Использование селекторов
- Разделение UI и бизнес-логики
- Оптимизация с помощью `React.memo` и `useCallback`
