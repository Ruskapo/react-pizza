 React Pizza

Полнофункциональное SPA-приложение для заказа пиццы, реализованное на React и TypeScript.
В проекте реализованы асинхронные запросы, фильтрация, маршрутизация, управление состоянием и оптимизация производительности.

Установка и запуск
npm install
npm start


Приложение будет доступно по адресу:

http://localhost:3000

Сборка production-версии
npm run build


Сборка будет создана в папке build/.
